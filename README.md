I developed a chatbot-based desktop application using Python Tkinter that allows users to search device inventory data stored in an Excel file. The chatbot guides the user through different search options such as model number, serial number, location, and invoice number. Based on user input, the system filters the dataset using Python and displays the results dynamically in a tabular format using a separate window. The project integrates GUI design, data handling using pandas, and event-driven programming.
